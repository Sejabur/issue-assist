      Info & User Manual

      Platform Overview

      The issue-assist Dashboard securely synchronizes all your data over the local network using a local JSON file, providing a fast, real-time environment for multiple users without relying on an external cloud server.

      Features & Functionality

      • Real-Time Local Sync: Changes are instantly synchronized locally to all active sessions to prevent data collisions.
      • Urgent Pings Sidebar: Important alerts are permanently visible in the sidebar. You can ping specific rows to immediately draw attention to them.
      • Bulk Actions & Undo: Select multiple rows to bulk edit or delete. Mistakes can be easily reverted using the Undo/Redo buttons in the header.
      • Dark & Light Mode: Seamlessly toggle between dark and light themes using the icon in the header.
      • Audit Logs: Every action (login, logout, modification, operator creation) is tracked with timestamp and username.
      • CSV Export: Back up the entire database instantly.

      Capabilities & Limits

      • Administrator: Full access to add, edit, and delete records. Can manage Operators, view full Audit Logs, and access Backup features.
      • Operator: Restricted access. Can create or edit records. Operators cannot delete records, access backups, or view settings.

      Security Warnings & Cautions

      • Data Integrity: IDs must be strictly unique numbers. Duplicate IDs will be aggressively rejected by the system.
      • Session Management: Always use the Log Out button when leaving your terminal to ensure audit logs capture session termination accurately.
      • Admin Credentials: Never share Admin credentials. Use the Settings panel to provision isolated Operator accounts for staff.
      • Rate Limits: To prevent abuse, the API proxy limits rapid successive modifications (Max 10 changes per 2 minutes per client).
